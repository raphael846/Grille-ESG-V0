Service cartographique fiable (recommandé — clé Geoapify gratuite)

Sans clé, la page utilise les serveurs OpenStreetMap publics, souvent saturés (quota par adresse IP). Avec une clé Geoapify (gratuite, 2 min : myprojects.geoapify.com → « Sign up » → nouveau projet → copier la clé ; 3 000 requêtes/jour offertes ≈ 200 analyses, sans carte bancaire), la recherche des lieux et les itinéraires piétons passent par leur infrastructure — mêmes données OpenStreetMap, sans saturation. La clé reste enregistrée uniquement dans votre navigateur. Si votre exemplaire du fichier a été fourni avec une clé intégrée, il n'y a rien à saisir.

Vérification IA du résultat (optionnel — Claude Haiku)

La clé reste enregistrée uniquement dans votre navigateur (jamais dans ce fichier ni dans le PDF). Créez-en une sur console.anthropic.com — pensez à y fixer une limite de dépense. Si le champ est vide, le rapport est généré sans vérification IA. Coût : ≈ 0,2 centime par rapport.

Tout se passe dans votre navigateur : géocodage, recherche de l'espace vert le plus proche et itinéraire piéton via OpenStreetMap, carte de preuve dessinée avec l'itinéraire réel, PDF généré et téléchargé. Aucune installation, aucun serveur.

Pour une preuve « capture d'écran Google Maps », utiliser la commande /s6 <adresse> dans Claude Code (le navigateur seul ne peut pas capturer Google Maps, par sécurité).
