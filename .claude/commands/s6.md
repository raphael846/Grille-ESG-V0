Génère le rapport S6 pour l'actif : $ARGUMENTS

1. Lis `skills/s6-biodiversite/SKILL.md` et respecte ses règles (notamment :
   ne jamais promettre d'intégrer une capture prise avec l'outil navigateur).
2. Dépendances (n'installe que si l'import échoue — gain de temps) :
   `python3 -c "import reportlab, PIL, staticmap, playwright" 2>/dev/null || pip install reportlab pillow staticmap playwright`
3. Lance le pipeline automatique :
   `python3 skills/s6-biodiversite/scripts/s6_auto.py "<adresse>" --out S6_<actif>.pdf`
   Si l'utilisateur a fourni un locataire/enseigne (ou si l'adresse semble
   vague, demande-lui-en un), ajoute `--locataire "Nom"`.
4. Relaie à l'utilisateur le niveau de preuve utilisé (ligne « Preuve : … »),
   envoie-lui le PDF, et vérifie la cohérence du résultat (espace vert retenu
   plausible, distance raisonnable). En cas d'échec réseau, explique que
   l'environnement bloque la sortie réseau et propose les alternatives du
   SKILL.md.
5. **Contrôle IA (si présent dans la sortie)** : le script imprime une ligne
   « Contrôle IA : … » uniquement quand une clé OpenAI (`OPENAI_API_KEY`) est
   configurée. **Le script ne modifie jamais le score de lui-même ; toute
   modification passe par l'accord explicite de l'utilisateur.** Comportement :
   - « Contrôle IA : confirmé » → le mentionner brièvement, rien de plus.
   - « Contrôle IA : DOUTE — <raison> » → signaler clairement le doute, puis :
     - **Si une « Alternative RÉELLE proposée : <nom> » est affichée** →
       proposer à l'utilisateur de retenir cet espace à la place (le 4/4 est
       conservé) : « L'IA doute de X (<raison>) mais Y, réel, valide aussi le
       critère. Je remplace par Y ? ». **Seulement s'il accepte**, relancer la
       même commande avec `--force-green-space "<nom>"`.
     - **Si « Aucune alternative valable »** → demander s'il faut baisser le
       score : « Aucun autre espace vert valable trouvé. Je passe le S6 à
       0/4 ? ». **Seulement s'il accepte**, relancer avec `--force-score 0
       --override-note "<raison>"`.
     - Ne JAMAIS proposer un espace qui n'apparaît pas dans les candidats réels
       du script : aucune invention.
   - « Contrôle IA : indisponible » → le dire en une ligne, sans bloquer.
   - Aucune ligne « Contrôle IA » → il n'y a pas de clé : comportement normal.
